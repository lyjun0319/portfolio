TITLE LABEL / INPUT[TYPE="CHECKBOX"] / LABEL INPUT STYLE

타이틀 label 과 함께 input[type=check] 사용법

  - label의 스타일은 타이틀 기본값으로 지정되어 있습니다.

  - input[type=check] 경우 label로 감싸야 합니다.

  - 기본속성label 의 스타일이 input[type=check]를 감싸는 label과 스타일 충돌이 나기 때문에 input[type=check]를 감싸는 label에 form-check-label클래스를 지정하여 스타일 충돌을 제재했습니다.

  - form-check-inline 타이틀 label과 동일 자식으로 label스타일속성중 vertical / padding / display의 속성값을 동일하게 조정하였습니다.


<div class="row">
	<div class="col-12">
		<div class="form-group">
			<label for="label03">성별</label>
			<div class="form-check-inline">
				<label class="form-check-label">
					<input type="radio" name="a" class="form-check-input" checked="">남성
				</label>
			</div>
			<div class="form-check-inline">
				<label class="form-check-label">
					<input type="radio" name="a" class="form-check-input">여성
				</label>
			</div>
		</div>
	</div>
</div>
